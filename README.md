<h1 align = "center"> Pokedex </h1>
<h2> Description </h2>
<p>Pokedex is a small application that shows a list of pokemons using pokeapi.</p>
<h2>Tech Stack</h2>
<ul>
    <li>Kotlin</li>
    <li>MVVM architecture</li>
    <li>Hilt for dependency injection.</li>
		<li>Retrofit</li>
		<li>Coroutines</li>
		<li>Glide</li>
</ul>
